웹 개발 언어 및 도구 검토

고려 사항: 모바일 우선 주의 (Mobile First Design)

  • 반응형 웹 디자인 (Responsive Web Design)
    • 핵심 기술 : 가변 그리드(fluid grid), 유연한 이미지(flexible images), 미디어 쿼리(media query)
    • Imagemin (이미지 압축 기술)
    • WebP와 같은 이미지 형식 사용
  • 프로그레시브 웹 앱 (Progressive Web App, PWA)
  • CSS 방법론 : BEM - 블록(Block), 요소(Element), 수식어(Modifier)

프로그래밍 언어

  • HTML5 (HyperText Markup Language)
  • CSS3 (Cascading Style Sheets) + (레벨4 모듈)
  • 자바스크립트 (JavaScript) + jQuery
